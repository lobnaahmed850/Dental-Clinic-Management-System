A JavaFX-based desktop application for managing a dental clinic.
The system allows managing doctors, patients, appointments, payments, and prescriptions.

## Features
- Add, update, and delete patients and doctors
- Schedule and manage appointments
- Detect appointment conflicts automatically
- Process payments linked to appointments
- Add prescriptions to appointments
- Sort appointments by date and time
- GUI built using JavaFX (FXML + Controllers)
- Demonstrates OOP concepts and exception handling

## Technologies Used
- Java
- JavaFX
- Maven
- NetBeans IDE

## OOP Concepts Demonstrated
- Encapsulation
- Inheritance
- Polymorphism
- Interfaces and abstract classes
- Custom exceptions

## How to Run
1. Open the project in NetBeans
2. Make sure JavaFX is configured
3. Run the main application
